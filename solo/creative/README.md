# Creative Portfolio Template

A single-layout portfolio starter for graduate school and professional development. **No coding tools required** — edit everything on github.com in your browser.

**Preview this layout:** [Creative demo](https://loganmjones.github.io/PortfolioTraining/themes/creative.html)

---

## Start here

1. Click the green **Use this template** button above → create a repo named `my-portfolio`.
2. **Settings** → **Pages** → Source: **main** branch, **/ (root)** → Save.
3. Open `index.html` — the blue **fill checklist** box lists every photo, PDF, and text block you need.
4. Search `EDIT HERE` and `ADD` to replace placeholders.
5. Colors: edit `css/creative.css` — only the hex codes at the top.

**Full setup guide:** [https://loganmjones.github.io/PortfolioTraining/guide.html](https://loganmjones.github.io/PortfolioTraining/guide.html)

**Other layouts:** [https://loganmjones.github.io/PortfolioTraining/](https://loganmjones.github.io/PortfolioTraining/)

## What you need to fill out Creative

Everything below completes the **basic template** — no extra sections required.

**AI shortcut:** Copy this entire checklist into Cursor or another agentic coding AI. Ask it to prompt you for each text block, image, PDF, and video, then implement the changes in `index.html` and tell you what to upload to `assets/`.

### Text

- Name and short intro (mosaic tiles)
- Gallery intro line
- 3 project tiles (title, description)
- Contact email, LinkedIn, GitHub

### Photos (upload to `assets/`)

**12 images total** — JPG recommended, ~500 KB each after compression.

| Qty | Subject | Min size | Filename |
|-----|---------|----------|----------|
| 1 | Mosaic intro tile | 400×400 px (1:1 square) | `assets/photo.jpg` |
| 1 | Large mosaic hero tile | 1200×600 px (2:1 wide) | `assets/hero.jpg` |
| 6 | Gallery grid (gallery-1 through gallery-6) | 400×400 px (1:1 square) | `assets/gallery-1-1.jpg … assets/gallery-1-6.jpg` |
| 1 | Featured project banner | 800×450 px (16:9 wide) | `assets/project-1.jpg` |
| 2 | Secondary project tiles | 400×400 px (1:1 square) | `assets/project-a-1.jpg … assets/project-a-2.jpg` |
| 1 | Media row still image | 800×450 px (16:9 wide) | `assets/process.jpg` |

### PDFs

_No PDFs required._

### Video

- 1× — Media row — reel, demo, or timelapse embed
