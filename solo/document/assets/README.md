# Assets folder — put your media files here

Upload these types of files via GitHub (Add file → Upload files):

| File | Example name | Used for |
|------|--------------|----------|
| Headshot | `photo.jpg` | Profile photo |
| Hero / banner | `hero.jpg` | Large top-of-page image |
| Project images | `project-1.jpg`, `project-featured.jpg` | Screenshots, posters, charts |
| Gallery | `gallery-1.jpg` … `gallery-6.jpg` | Photo grids |
| CV / papers | `cv.pdf`, `paper.pdf`, `slides.pdf` | Download links |
| Audio | `interview.mp3` | Optional audio clips |

**Before uploading:** compress images at https://squoosh.app (free, no account).

**Privacy:** only upload files you're allowed to share publicly. See guide.html#privacy
