# Editorial Portfolio Template

A single-layout portfolio starter for graduate school and professional development. **No coding tools required.** Edit everything on github.com in your browser.

**Preview this layout:** [Editorial demo](https://loganmjones.github.io/PortfolioTraining/themes/editorial.html)

---

## Start here

1. Click the green **Use this template** button above → create a repo named `my-portfolio`.
2. **Settings** → **Pages** → Source: **main** branch, **/ (root)** → Save.
3. Open `index.html`. The blue **fill checklist** box lists every photo, PDF, and text block you need.
4. Search `EDIT HERE` and `ADD` to replace placeholders.
5. Colors: edit `css/editorial.css`. Change only the hex codes at the top.

**Full setup guide:** [https://loganmjones.github.io/PortfolioTraining/guide.html](https://loganmjones.github.io/PortfolioTraining/guide.html)

**Other layouts:** [https://loganmjones.github.io/PortfolioTraining/](https://loganmjones.github.io/PortfolioTraining/)

## What you need to fill out Editorial

Everything below completes the **basic template**. No extra sections required.

**AI shortcut:** Copy this checklist into Cursor, Copilot, or another coding assistant. Ask it to request each text block, image, PDF, and video from you, update `index.html`, and tell you what to put in `assets/`.

### Text

- Name, tagline, about (2–3 paragraphs + optional quote)
- Experience: 2 entries
- Featured project: title, description, tags
- 2 more projects (title, description, links)
- Contact email, LinkedIn, GitHub

### Photos (upload to `assets/`)

**9 images total.** JPG recommended, about 500 KB each after compression.

| Qty | Subject | Min size | Filename |
|-----|---------|----------|----------|
| 1 | Banner: campus, lab, or research visual | 1600×900 px (16:9 wide) | `assets/hero.jpg` |
| 1 | Headshot in about section | 400×400 px (1:1 square) | `assets/photo.jpg` |
| 1 | Featured project image | 800×450 px (16:9 wide) | `assets/project-featured.jpg` |
| 2 | Thumbnails on “More projects” cards | 600×340 px (16:9 wide) | `assets/project-2-1.jpg … assets/project-2-2.jpg` |
| 4 | Gallery grid (rename gallery-1 through gallery-4) | 300×300 px (1:1 square) | `assets/gallery-1-1.jpg … assets/gallery-1-4.jpg` |

### PDFs

| Document | Filename | Notes |
|----------|----------|-------|
| CV | `assets/cv.pdf` | Hero download button |

### Video

- 1×: Featured project: YouTube/Vimeo embed

| File | Purpose |
|------|---------|
