Setup Guide

No coding experience needed. Everything happens on github.com in your browser.

1 Copy this template

  1. Go to the template repo on GitHub.
  2. Click the green Use this template button → Create a new repository.
  3. Name it something like my-portfolio.
  4. Click Create repository.

2 Turn on GitHub Pages (free hosting)

  1. In your new repo, click Settings (top tab).
  2. Click Pages in the left sidebar.
  3. Under Build and deployment → Source: pick Deploy from a branch.
  4. Branch: main · Folder: / (root) → click Save.
  5. Wait 1–2 minutes. Your URL appears at the top of the Pages settings page:
    https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/
⚠️ Important Do NOT preview your site by clicking index.html inside the repo — that shows unstyled HTML. Always use your GitHub Pages URL from Settings → Pages.

3 Pick a layout

Open your live site URL. You'll see 6 layout options. Click through them using the black bar at the top. Pick the one you like best — you'll only customize that one file.

4 Edit your content on GitHub

  1. Go to your repo on github.com.
  2. Click themes folder → click your chosen file (e.g. editorial.html).
  3. Click the pencil icon (✏️ Edit this file) in the top-right.
  4. Press Cmd+F (Mac) or Ctrl+F (Windows) and search: EDIT HERE
  5. Change each labeled spot to your real information.
  6. Scroll to the bottom → click Commit changes.

5 Add your photo

  1. In your repo, click the assets folder.
  2. Click Add file → Upload files.
  3. Drag in your photo (name it something simple like photo.jpg).
  4. Click Commit changes.
  5. Go back to your theme file and search for ADD YOUR PHOTO HERE.
  6. Follow the instructions in that comment to uncomment the <img> line and update the filename.

6 Change your colors

  1. Click the css folder in your repo.
  2. Click the file matching your layout (e.g. editorial.css for Editorial layout).
  3. Click the pencil icon to edit.
  4. At the very top you'll see a section labeled CHANGE YOUR COLORS HERE.
  5. Only edit the hex color codes (like #2563eb). Don't change anything else unless you know CSS.
  6. Commit changes.

Need a color? Google "color picker" or use htmlcolorcodes.com.

7 Clean up when done

  • Delete the yellow trainer tip boxes from your HTML file (search for trainer-tip).
  • Update your links — search for EDIT LINK HERE and replace # with real URLs.
  • Hard-refresh your live site: Cmd+Shift+R or Ctrl+Shift+R.

Search cheat sheet

Use Cmd+F / Ctrl+F in your theme file to find these:

EDIT HERE — NAME
Your full name
EDIT HERE — TAGLINE
One-sentence description of you
EDIT HERE — ABOUT
Your bio paragraphs
EDIT HERE — EMAIL
Your email address
ADD YOUR PHOTO HERE
Where to put your headshot
EDIT LINK HERE
LinkedIn, GitHub, project URLs

Using GitHub Copilot

Copilot can rewrite your bio, suggest project descriptions, or help fix links. Open your file for editing on GitHub, then use Copilot Chat and try:

  • "Rewrite the about section for a PhD application in neuroscience"
  • "Make this experience bullet more professional"
  • "What does EDIT HERE — EMAIL mean in this file?"
