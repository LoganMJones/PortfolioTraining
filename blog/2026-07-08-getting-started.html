Getting started with a plain HTML/CSS portfolio

2026-07-08

Plain HTML and CSS are a strong foundation for a grad-school portfolio because they produce pages that are fast, predictable, and easy to audit for accessibility. Start by making your resume the homepage, add a projects index that links to case studies, and keep the blog simple so you can focus on writing.

Tips

  • Use semantic tags (header, main, article, section)
  • Provide a print stylesheet for a downloadable resume
  • Store images in /assets/images and optimize them (srcset, WebP)
