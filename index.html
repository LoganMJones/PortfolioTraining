How it works

  1. Preview layouts on this site — click any card or use the black bar on each page.
  2. Use template on GitHub for the layout you chose (one repo per layout — no bloat).
  3. Enable Pages in your new repo — see the setup guide.
  4. Edit index.html — search for EDIT HERE.
