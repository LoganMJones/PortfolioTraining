How it works

  1. Preview on this site (cards or the black bar on each page).
  2. Use the template for that one layout on GitHub (one repo, no extras).
  3. Turn on Pages in the new repo. Details in the setup guide.
  4. Edit index.html. Search for EDIT HERE.
