How customization works

  1. Pick one layout using the cards above (or the black bar on each page).
  2. Edit on GitHub — click the pencil icon on your chosen themes/____.html file.
  3. Search for ADD — images, video, PDF, gallery, and audio slots are labeled.
  4. Read the privacy section before publishing personal info.
  5. Delete yellow & red boxes when you're done — training helpers only.
