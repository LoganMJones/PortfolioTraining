How customization works

  1. Pick one layout using the cards above (or the black bar on each page).
  2. Edit on GitHub — click the pencil icon on your chosen themes/____.html file.
  3. Search for EDIT HERE — every spot you need to change is labeled.
  4. Change colors in the matching css/____.css file (instructions at the top).
  5. Add your photo where you see ADD YOUR PHOTO HERE.
  6. Delete the yellow boxes when you're done — those are just training helpers.
